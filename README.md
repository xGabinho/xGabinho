## 👋 Sobre mí

Soy desarrollador Full Stack con base en **Medellín, Colombia**, actualmente formándome en **ADSO en el SENA**. Me apasiona construir productos web que combinen buena arquitectura con una experiencia de usuario sólida. Cada proyecto es una oportunidad de aprender algo nuevo.

- 🔭 Trabajando en **PCMatch** y **Karomi**
- 📚 Profundizando en arquitecturas limpias y patrones de diseño
- 🎯 Objetivo: consolidar mi stack y aportar valor real en cada proyecto
- 🎮 Fuera del IDE: videojuegos, mis mascotas y tiempo con amigos
